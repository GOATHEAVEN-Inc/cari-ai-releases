# 카리AI — 다운로드

범용 AI 어시스턴트이자 생활 기기 조작 엔진, **카리AI**의 공개 배포 리포입니다.
(제품 소스는 비공개 리포에서 관리하며, 이 리포는 **설치본 배포 + 다운로드 랜딩 페이지** 용도입니다.)

## 다운로드

- **다운로드 페이지**: https://goatheaven-inc.github.io/cari-ai-releases/
- **최신 릴리스**: [Releases](https://github.com/GOATHEAVEN-Inc/cari-ai-releases/releases/latest)
  - macOS: `CariAI.dmg`
  - Windows: `CariAI.exe`

터미널 없이 일반 앱처럼 더블클릭으로 설치합니다. (미서명 빌드라 첫 실행 시 OS 경고가 뜰 수 있습니다.)
